# Hole Wizards

A simple ASCII graphical dungeon crawling role playing game written in Python.

## Features

- Text interface for user commands and feedback
- ASCII map showing explored areas
- Statistics window with multiple pages (player, enemy, inventory, chest)
- Log and command interface

## Getting Started

1. Clone the repository
2. Run the main Python script (to be implemented)

## UI Overview

- **Top Left:** Map (player, monsters, walls, doors, chests)
- **Bottom:** Text log and command input
- **Top Right:** Stats and inventory pages
