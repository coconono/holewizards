# png animation studio 

pygame tool used to create frame by frame animations

loads png files into a 2d view

png files files can be arranged
- moved around the viewscreen
- flipped on the x or y axis
- layered on top of each other

once desired arrangement is achieved, the user can save that as a frame
- frames have a name
- saved to a .frame file

frames live in a frame library
- libraries have a name
- this is saved a .library file

frames are arranged into timelines
- timelines have a name
- timelines can be played back
- saved to .timeline files
